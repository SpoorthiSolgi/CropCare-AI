
# CropCare AI – Multimodal Plant Disease Diagnosis Agent

This repository contains the full code and demo notebook for the CropCare AI agent used in the Kaggle Agents Capstone Project.

## Contents
- `notebooks/demo.ipynb` – main demo notebook  
- `src/` – agent modules  
- `assets/` – images / thumbnails  
- `data/` – memory store  
- `requirements.txt`

## How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Open the demo notebook:
   ```
   jupyter notebook notebooks/demo.ipynb
   ```

## Description
CropCare AI analyzes a leaf image, identifies possible diseases, and generates treatment recommendations.
